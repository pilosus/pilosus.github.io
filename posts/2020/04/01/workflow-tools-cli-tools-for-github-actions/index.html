workflow-tools: CLI tools for GitHub Actions

01-04-2020 18:10 | by Vitaly Samigullin | in Blog | 87 words | 1 min to read
tags: opensource automation microservice

Microservice architecture may have dozens and dozens of lookalikes services that require similar CI/CD workflows. With infrastructure as code approach taken by the GitHub Actions, why not using workflows generation? Provisioning repository for a new microservice may also be automated.

I’ve just opensourced an internal tool we use here at ANNA to automate setting up new microservices. workflow-tools is a collection of CLI tools used to:

  • generate GitHub Actions workflows from a Jijna2-template
  • set GitHub secrets (used by the workflows)
